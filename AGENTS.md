# Instructions for Codex

## UI design style

When creating or modifying any website, app, dashboard, landing page, or UI component:

- Do not create bubble-style UI.
- Do not wrap buttons, text, images, icons, inputs, navigation items, or cards in decorative rounded containers.
- Avoid pill-shaped wrappers, floating bubbles, unnecessary badges, excessive rounded corners, nested cards, glassmorphism blobs, and soft shadow boxes.
- Prefer clean spacing, alignment, typography, layout rhythm, simple sections, and minimal borders.
- Use rounded corners only when they serve a real functional purpose.
- Before finishing, review the UI and remove unnecessary bubbles, pills, rounded wrappers, and decorative containers.
