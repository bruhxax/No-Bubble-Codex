# No Bubble UI

When designing or editing UI, avoid the “bubble UI” look.

## Hard rules

Do not add decorative rounded containers around:

- buttons
- text blocks
- images
- icons
- form fields
- navigation items
- cards
- sections
- unrelated UI elements

Avoid:

- pill-shaped wrappers
- floating bubbles
- excessive border-radius
- nested cards
- outlined boxes around every small element
- soft shadow containers around small UI pieces
- badges everywhere
- glassmorphism blobs unless explicitly requested

## Preferred style

Use:

- clean spacing
- strong alignment
- typography hierarchy
- grid/layout rhythm
- simple sections
- subtle dividers only when useful
- minimal borders
- restrained border radius only for functional surfaces

## Final check

Before finishing any UI task:

1. Look for unnecessary bubbles, pills, wrappers, rounded boxes, or decorative containers.
2. Remove them.
3. Keep only containers that have a clear functional purpose.
4. If a rounded container is necessary, make it subtle.

## Forbidden patterns

NEVER use:
- rounded floating cards everywhere
- capsule buttons
- giant border radius
- floating UI islands
- neumorphism
- glassmorphism bubbles
- dashboard cards for every section
