# NO-BUBBLE UI Skill

Minimal UI design rules for Codex and AI agents.  
Prevents bubble-style interfaces, excessive rounded cards, floating containers, and overdecorated UI.

---

# NO-BUBBLE UI Skill (RU)

Минималистичный набор правил для Codex и AI-агентов.  
Запрещает bubble-style интерфейсы, лишние карточки, плавающие контейнеры и перегруженный UI.

---

# Features / Возможности

## English

- Removes bubble-style UI patterns
- Prevents excessive rounded corners
- Avoids floating cards and capsule buttons
- Encourages flat minimal interfaces
- Improves visual cleanliness
- Works for:
  - websites
  - dashboards
  - SaaS
  - mobile apps
  - landing pages
  - admin panels

## Русский

- Убирает bubble-style UI
- Запрещает чрезмерные скругления
- Убирает плавающие карточки и капсульные кнопки
- Продвигает flat/minimal дизайн
- Делает интерфейс чище
- Подходит для:
  - сайтов
  - dashboard
  - SaaS
  - мобильных приложений
  - лендингов
  - админ-панелей
